# CF_AI_CHAT

An AI chat application that remembers context of the conversation using Cloudflare's worker AI on llama 3.3

# Prerequisites:
 - Node.js 20+
 - Cloudflare account (free tier works)

# Setup Instructions

```bash
# Install dependencies
npm install

# Login to Cloudflare
npx wrangler login

# Start dev server
npm run dev
